## Запуск через Docker Compose (Лабораторна робота №2)

Усі сервіси (база даних, застосунок, nginx) контейнеризовані та можуть бути запущені однією командою. База даних використовує Docker Volumes для збереження стану між перезапусками. Усі контейнери ізольовані у власній мережі `lab_net`.

**Інструкція з запуску:**
1. Впевніться, що на машині встановлено `docker` та `docker-compose-plugin`.
2. Перейдіть у корінь репозиторію.
3. Виконайте команду:
   ```bash
   docker compose up -d --build
